exemplo acceptedlanguagesui

A biblioteca acceptedlanguagesui insere um elemento na página, no topo da página, por padrão, convidando o visitante a visualizar esta página em uma linguagem que é preferido por eles. Altere o idioma do seu navegador para ver como isso afeta o idioma que se ofereceu para você em carga.

Se você não vê o convite no topo desta página, é porque você está visualizando a página em uma língua que você já preferir.

Se o visitante seleciona Não, a decisão será lembrado e não será solicitado novamente. Se você gostaria de tentar novamente claros seus navegadores Armazenamento local.

Se o visitante seleciona Sim, eles são redirecionados para o URL no atributo href do elemento <link> alternativo no <head> da página correspondente ao idioma.

Note: This page is viewable in en-AU, es, pt.
