acceptedlanguagesui example

The acceptedlanguagesui library inserts an element into the page, at the top of the page by default, inviting the visitor to view this page in a language that is preferred by them. Change your browser's language to see how it affects which language gets offered to you on load.

If you don't see the invitation at the top of this page, it's because you're viewing the page in a language you already prefer.

If the visitor selects No, the decision will be remembered and they won't be asked again. If you'd like to try again clear your browsers Local Storage.

If the visitor selects Yes, they are redirected to the URL in the href attribute of the alternate <link> in the <head> of the page matching the language.

Note: This page is viewable in en-AU, es-419, pt-BR.
